# Changelog

Todas as mudanças relevantes do projeto Cactus Labs serão documentadas neste arquivo.

O formato é baseado no [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adota o [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2026-03-12

### ✨ Adicionado

- Dashboard completo de performance com métricas reais do GitHub
- Sistema de autenticação confiável via OAuth GitHub utilizando o Supabase Auth
- Análise aprofundada de commits e Pull Requests através de IA (integração Groq + gpt-oss-120b)
- Exportação dinâmica de relatórios de métricas em formatos PDF e Excel
- Heatmap de contribuições visualmente idêntico ao estilo clássico do GitHub
- Gráficos de atividade interativos implementados de fácil legibilidade com Recharts
- Suporte a organizações do GitHub para rastrear performance granular de times e contribuidores
- Landing page pública modernizada com manifesto oficial e seção conceitual de pricing estruturada
- Ajuste e definição de tema claro/escuro nativamente alternável (implementação radix + shadcn/ui)
- Interface integral em pt-BR focada na demanda nacional
